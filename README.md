# SportiGo 🏏⚽🏸🏀

SportiGo is a Sports Equipment E-Commerce Website developed using Flask, Python, MySQL, HTML, CSS, Bootstrap, and JavaScript.

## Features

* Admin Login & Registration
* User Login & Registration
* Category Management
* Product Management
* Product Details Page
* Add to Cart
* Buy Now
* Checkout System
* My Orders
* Cancel Orders
* About Us Page
* Contact Us Page

## Technologies Used

* Python
* Flask
* MySQL
* HTML
* CSS
* Bootstrap
* JavaScript
